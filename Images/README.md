# 📊 Bank Loan Analysis – Dashboard Images

This folder contains visual assets and screenshots from the **Bank Loan Analysis Power BI Dashboard**.

These images are used to showcase the key sections and insights of the interactive dashboard.

The bank logo used for branding and visual identity in the project.
